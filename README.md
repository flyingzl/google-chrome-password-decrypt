# Introduction

Get the saved password from Chrome on MacOS


# Usage  

```bash
Usage of ./hackChrome:
  -k string
    	keywords for website url
  -p string
    	profile path for  Google Chrome, you can check with chrome://version  (default "/Users/XXX/Library/ApplicationSupport/Google/Chrome/Default")
```
